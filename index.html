Welcome to Knowledge Cloud Research Group.

This group intends to perform research on Knowledge Search Engine with the same purpose as Google Knowledge Graph, WikiData, Wolfgram Alpha, etc. The basic technology that we are interested in is the Semantic Web technologies.

Join this Group

  • Subscribe to the group mail-list: http://tech.groups.yahoo.com/group/ccwiki_zju/
  • Participate the development: https://github.com/knowledge-cloud

Projects List

  • CCWiki: A Social Shopping Application based on Semantic Media Wiki and Semantic Mashup. More
  • sMash: A mashup engine that can harvest data from all kinds of online data APIs using the Semantic Web technologies. More
